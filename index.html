How to Use This System

1. Initial Setup (Monitor)

As a monitor, ensure your Firebase project is linked in `js/firebase-init.js` and that `style.css` and the JavaScript files are correctly placed. You will need to populate your Firestore with initial classes (e.g., S.1, S.2, etc.) in the `classes` collection. Add students to each class via the Monitor Dashboard.

2. Monitor Dashboard

Select Class & Date: Choose a class (e.g., S.1) and a date to view attendance records. Mark Attendance: For each student, select 'Present' or 'Absent'. Click 'Save Changes' to update the database. Add New Student: Enter a unique roll number and name for a new student, then click 'Add Student'. Remove Student: Use the trash icon next to a student's name for quick removal from a class, or use the "Permanently Remove Student" section to delete by name (be cautious, this is irreversible!).

3. Student Portal

Select Your Class: Choose your class (e.g., S.1). Select Date: Pick the date for which you want to check attendance. View Attendance: Click 'View Attendance' to see your status for that day.

4. Best Practices

Always refresh your browser if you encounter issues. Ensure network connectivity for real-time updates. For production, implement user authentication and stricter Firebase security rules.
